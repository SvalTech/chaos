# ChaosPrep 

ChaosPrep is a Single Page Application (SPA) designed as an advanced exam tracker for JEE, NEET, and competitive aspirants. It provides real-time task management, a focus timer with Picture-in-Picture (PiP) support, complex analytics for mock exams and error tracking, and a real-time multiplayer "Squad" accountability system. 

## Tech Stack
* **Frontend Core:** Vanilla JavaScript (ES6 Modules), HTML5, CSS3.
* **Styling:** Tailwind CSS (utility-first CSS framework).
* **Backend & Database:** Firebase Auth, Firebase Firestore (NoSQL).
* **Libraries & APIs:**
    * `Chart.js` (Canvas-based data visualization)
    * `SortableJS` (Drag-and-drop task reordering)
    * `Lucide Icons` (SVG icon rendering)
    * `YouTube Iframe API` (Background focus radio)
    * Screen Wake Lock API & Document Picture-in-Picture API.
