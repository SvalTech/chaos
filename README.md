# ChaosPrep 

ChaosPrep is a Single Page Application (SPA) designed as an advanced exam tracker for JEE, NEET, and competitive exam aspirants. It provides real-time task management, a focus timer with Picture-in-Picture (PiP) support, complex analytics for mock exams and error tracking, and a real-time multiplayer "Squad" accountability system.
